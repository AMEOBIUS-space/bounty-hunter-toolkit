# Contributing

Thank you for your interest! This project uses zero external dependencies — pure Python stdlib only.

## Submitting Changes

1. Fork the repository
2. Create a feature branch
3. Make your changes (zero pip dependencies only)
4. Run tests: `python -m pytest tests/ -v`
5. Open a Pull Request with a clear description

## Code Standards

- Python 3.10+
- Standard library only (no pip dependencies)
- 80%+ test coverage
- MIT License

## Reporting Issues

Open an issue with:
- Python version
- Steps to reproduce
- Expected vs actual behavior
